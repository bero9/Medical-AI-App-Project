# AI-Powered Medical Assistant for People with Disabilities 🏥 🤖

---

## English: Technical Stack & Models
This project leverages state-of-the-art Computer Vision models to provide real-time assistance:

### 🧠 AI Models Integrated:
* **YOLO (You Only Look Once):** High-speed real-time object detection for the blind.
* **MiDaS:** Monocular Depth Estimation to calculate the distance between the user and obstacles.
* **R-CNN:** Precise Optical Character Recognition (OCR) for reading medical prescriptions and labels.

### 🛠️ Infrastructure:
* **Backend:** Django (Python) framework for robust API management.
* **Frontend:** Flutter (Dart) for a seamless cross-platform mobile experience.
* **Deployment:** Ngrok for secure tunneling and real-time mobile-to-backend communication.

---

## Deutsch: Technischer Stack & KI-Modelle
Dieses Projekt nutzt modernste Computer-Vision-Modelle für die Echtzeit-Unterstützung:

### 🧠 Integrierte KI-Modelle:
* **YOLO (You Only Look Once):** Hochgeschwindigkeits-Objekterkennung in Echtzeit für Blinde.
* **MiDaS:** Monokulare Tiefenschätzung zur Berechnung des Abstands zwischen Benutzer und Hindernissen.
* **R-CNN:** Präzise optische Zeichenerkennung (OCR) zum Lesen von medizinischen Rezepten und Etiketten.

### 🛠️ Infrastruktur:
* **Backend:** Django (Python) Framework für robustes API-Management.
* **Frontend:** Flutter (Dart) für eine nahtlose plattformübergreifende mobile Erfahrung.
* **Deployment:** Ngrok für sicheres Tunneling und Echtzeit-Kommunikation zwischen Mobile und Backend.
